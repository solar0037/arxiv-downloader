# arxiv-downloader

arxiv-downloader is a command-line program to download papers from [arxiv.org](https://arxiv.org/).

## Installation

Download the latest release from [releases](https://github.com/solar0037/arxiv-downloader/releases) and install it with pip.

```shell
pip install arxiv_downloader-[version].whl
```

Alternatively, install from the repository.

```shell
pip install -e git+https://github.com/solar0037/arxiv-downloader#egg=arxiv-downloader
```

## Usage

- `arxiv-downloader [id]` - Download the paper that matchs the id.
- `arxiv-downloader [ids]` - Download papers that match the ids.
